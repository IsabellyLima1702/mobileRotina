# Rotina

## Descrição:
### Este projeto tem como objetivo oferecer rotinas prontas de cuidados para bebês e a opção para o usuário criar sua própria rotina. 

## Etapas:
### O projeto inclui algumas etapas: Escolha do interesse, criação da rotina (caso o usuário opte por não utilizar uma rotina pronta) e por último, a tela com todas as rotinas feitas, com a possibilidade de edição (caso o usuário opte por utilizar a rotina pronta, aparecerão as rotinas disponíveis + as criadas por ele).
